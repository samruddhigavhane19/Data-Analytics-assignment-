Consumer Sales Analytics for Apple Product Performance
Project Overview:
This project focuses on analyzing the Apple Global Sales Dataset to understand sales performance in the consumer electronics market. The analysis uses statistical methods, data cleaning, visualization, and simple machine learning techniques to study revenue patterns, product category performance, and customer behavior. Different charts were created to identify trends in monthly sales, category-wise revenue, and profit distribution. A prediction model was also applied to examine how selected variables influence business outcomes. The project helps demonstrate how data analytics can support decision-making in sales and product management.
Prediction Model:
Logistic Regression was applied to build a simple prediction model. Feature scaling was performed before training to improve model performance. The model was trained on 80% of the dataset and tested on 20% of the dataset. Accuracy was used to measure prediction performance.
Result:
The analysis showed clear patterns in revenue, category performance, and monthly sales trends within the Apple Global Sales Dataset. High-performing categories contributed more to total revenue and profit, while some transactions produced unusually high revenue values. The prediction model also achieved measurable accuracy, showing that selected features can help explain business outcomes.
Conclusion:
This project demonstrates how data analytics can be used to understand sales performance in global consumer electronics. Statistical analysis, visualizations, and prediction techniques helped identify important business trends in the dataset. The findings can support better decisions related to product focus, sales planning, and customer satisfaction improvement.
